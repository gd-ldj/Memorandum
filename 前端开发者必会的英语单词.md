## 前端开发者必会的英语单词
### git commit 日志相关单词
- feat `[fēt]`: 新增功能
- fix `[fiks]`: 修复bug
- docs `[doks]`: 文档改动
- style `[stīl]`: 样式改动
- test `[test]`: 测试用例改动
- refactor `riˈfaktər`: 重构、优化
- chore `[CHôr]`: 项目工程改动
